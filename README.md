# Chrome Dino

A Simple implementation of chorme dino (with archer :)) using ncurses.

### How to use:
1. Install Ncurses
   * On linux:
     * `sudo apt-get install libncurses5-dev libncursesw5-dev`
   * On Mac OS:
     * `brew install ncurses`
2. Compile
  * `gcc main.c -o game -lncurses`
3. Run
  * `./game`
4. Enjoy it :)
